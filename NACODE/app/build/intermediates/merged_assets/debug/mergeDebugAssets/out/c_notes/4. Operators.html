Logical Operators

Logical AND ( && ):Logical AND operator. If both the operands are non-zero, then the condition becomes true.

Logical OR ( || ):Logical OR Operator. If any of the two operands is non-zero, then the condition becomes true.

Logical NOT ( ! ) :Logical NOT Operator. It is used to reverse the logical state of its operand. If a condition is true, then Logical NOT operator will make it false.

Assignment Operators

The basic assignment operator is equal ( = ), which assigns the value of its right operand to its left operand.

Eg: = , += , -= , *= , /=.

Bitwise Operators

Bitwise operator works on bits and perform bit-by-bit operation. Arithematical operations are converted to bit-level which makes processing faster and saves power.
