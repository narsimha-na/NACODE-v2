Intput and Output

In C programming language provides many built-in functions which helps you to read and write the data. stdib is the standard C library for the input and output operations.Their are mainly three types of intput adn output functions,they are :

  • scanf() and printf()
  • puts() and gets()
  • getchar() and putchar()

scanf() and printf() functions

The standard header file contains these functions printf() and scanf() ,in which scanf() it is used read user input and it returns user input and printf() function is used to display ouput or result.

#include <stdio.h> void main() { int a; //scanf() used for taking user input scanf("%d",a); //printf() used for displaying result or output printf("The user Input : %d",a); }

NOTE : in the above program their is usage of '%d' in the program,that is known as the Format String . The Format string this is used to inform the scanf() and printf() which type of input or output is used respectively.

  • ' %d ' : for integer type data
  • ' %c ' : for character type data
  • ' %f ' : for float type data
  • ' %s ' : for string type data

getchar() and putchar() functions

In this getchar() is function which reads only one character at a time and return it as an integer.putchar() is function which is used to display result on to screen and it also display one chracter at a time.


gets() and puts() functions

In this gets() function reads a line from the stdin into a pointer until the newline or EOF (end of file) .puts() function writes the string str and a trailing newline to stdout.
