Structure of C Program

#include <stdio.h> int main() { /* Hello World program */ printf("Hello, World! \n"); return 0; }

Hello World Program in C

Output

Hello World!

Let us look at the program explanation :

  • " #include " : It is a preprocessor command, it is used for telling the compiler that to include 'stdio.h' ,it may be any any header file .
  • " int main() " : It is the main part of the program ,where the excution starts.
  • " /*...*/ " : These are the operands used by programmer ,these opearnds are ignored by compiler. These are known as comments
  • " printf("") " : This function of C helps us to display ouput or any of the data .
  • " return 0; " : The next line return 0; terminates the main() function and returns the value 0.
