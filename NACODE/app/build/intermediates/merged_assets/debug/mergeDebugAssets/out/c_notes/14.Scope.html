Scope in C programming Langauge

Scope in programming language is the region in which program where the varible can be declared and beyond that it cannot be accessed.Their are two types variables , they are :

  • Gloabal Variable

    Global variable are declared outside any functions, usually at top of program.they can be used by later blocks of code those variables are known as Global variables .

  • Local Variable

    Variables that are declared inside a function or block are local variables.The scope of local variables will be within the function only.These variables are declared within the function and can't be accessed outside the function.
