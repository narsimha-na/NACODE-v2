Passing Arugments to a function

Arugments are passed during the function call , these varibles are called Foraml parameters . Passing of arguments to function are of two types ,they are :

  • Call by Value : method of passing arguments to a function copies the actual value of an argument into the formal parameter of the function. In this case, changes made to the parameter inside the function have no effect on the argument.

  • Call by Reference : method of passing arguments to a function copies the actual value of an argument into the formal parameter of the function. In this case, changes made to the parameter inside the function have no effect on the argument.
