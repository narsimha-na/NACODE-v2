Arrays

Arrays are special variables which can hold more than one value under the same variable name, organised with an index.An array is defined as finite ordered collection of homogenous data, stored in contiguous memory locations.

Declaring Array

Like any other variable, arrays must be declared before they are used. General form of array declaration is,

dataType variableName[size]; 

//Example
int na[30];

Initializing Arrays

Array can initialized in two ways at runtime or compile time and arrays can be of differnt types , they are :

  • One Dimensional Array

    It is the basic of the array , in which we have only one index for the array.

    Initializing a Arrya :

    nacode[0]=1;
nacode[1]=2;
nacode[2]=3;
nacode[3]=4;

    Example :

    #include<stdio.h> void main(){ int i=0; // Declaring of the array int na[3]; //Initializing of the array na[0]=1; na[1]=2; na[2]=3; for(i=0;i<=3;i++){ //Accessing of array printf("%d \t",na[i]); } }

    Output :

    1 2 3

  • Two Dimensional Array

    Two Dimensional Array is a of a list of one dimensional arrays, it is the simplest form of the multidimensional array.It should have to two indeed value .

    Initializing :

    int arr[4][3]={{1,2,3},{2,3,4},{3,4,5},{4,5,6}};

    Example :

    #include int main(){ int i=0,j=0; int na[2][2]={{1,2},{2,3}}; //traversing 2D array for(i=0;i<=2;i++){ for(j=0;j<=2;j++){ printf("na[%d] [%d] = %d \n",i,j,na[i][j]); } } return 0; }

    Output : 

    na[0][0] = 1 
na[0][1] = 2
na[1][0] = 2 
na[1][1] = 3
