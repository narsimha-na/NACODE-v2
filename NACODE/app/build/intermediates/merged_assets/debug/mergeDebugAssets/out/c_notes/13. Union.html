Unions

C Union is also like structure, i.e. collection of different data types which are grouped together.he syntax to declare/define a union is also similar to that of a structure. The only differences is in terms of storage. In structure each member has its own storage location, whereas all members of union uses a single shared memory location which is equal to the size of its largest data member.

Syntax :

union tag_name
{
data type var_name1;
data type var_name2;
data type var_name3;
};

Accessing a Union Member

    		
union test
{
    int a;
    float b;
    char c;
}t;

t.a;    //to access members of union t
t.b;     
t.c;

Example :

#include <stdio.h> #include <string.h> union employee { int id; char name[50]; }na; //declaring na variable for union void main( ) { na.id=16951A0564; strcpy(na.name, "Kartik Joshi"); //printing first employee information printf( " id : %d\n", na.id); printf( " name : %s\n", na.name); }
