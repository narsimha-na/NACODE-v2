Variables

A Variable in simple terms is a storage place which has some memory allocated to it,variable used to store some form of data. Different types of variables require different amounts of memory.

Variable Types

  • char : A character constant is created by enclosing a single character inside single quotation marks.

  • int :An integer is a numeric constant (associated with number) without any fractional or exponential part.

  • float : A floating point constant is a numeric constant that has either a fractional form or an exponent form.

  • double : It is the double of the floating value.

    Rules on writing the varible names

  • A variable name can consist of Capital letters A-Z, lowercase letters a-z, digits 0-9, and the underscore character.
  • The first character must be a letter or underscore.
  • C keywords cannot be used as variable names.
  • Special characters like #, $ are not allowed.
