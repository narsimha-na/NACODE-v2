Decision Making

In this Decision making section , when a developer writing code he spifices one or more conditions to be checked or tested by the program , if that following condition is true then it excutes then a statement is executed or the other statement is executed if it false.

C programming langauge provides the following types of Decision making statement :

  • if statement

    In this expression ,if the condition is true then the statement inside the 'if ' condition will get excuted otherwise it is skipped.

    Syntax :

    if(condition){
	
  statement;

}

  • if else statement

    In this expression , if statement can be followed by an optional else statement. If the first condition faills ,then the else statement is executed.

    Syntax :

    if(condition){
	
	statement;

}else{
	
	statemnet;
}
