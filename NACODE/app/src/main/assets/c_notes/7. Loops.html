Looping

In the world of programming code efficiency is very important, some times we repeat the same piece of code again and agian , due to which execution becomes slow.So by these loop statements we can repeat the same code again and again without writing it agian.

Their are different types of loops , they are :

  • while loop

  • for loop

  • nested loop

  • do while loop

for loop

This is a type of for loop in which ,we have to specify the number of times the loop as to be executed then it will executed that number number of times.

Syntax :

for(initialization;condition;increment/decrement){
	
	statement;
}

while loop

This is a type of loop in which , we have to repeat the given statement until the given condition is true.it first test the condition before executing loop.

Syntax :

while(condition){
	
	statement;
	increment/decrement;
}

do while loop

This is the type of the loop in which the loop is executed first and then the condition is executed. do statements checks into that loop is executed and whereas while statement checks into that the given condition is true

do{
	......
	statements;
	......
}
